# Gaussian Scene Workbench 0.2.7

## English

- Identifies this build as `Gaussian Scene Workbench (Legacy HTML)` in both the browser document title and Electron window title.
- Uses the dedicated Windows AppUserModelID `Fubukisama.GaussianSceneWorkbench.LegacyHtml` so Windows can distinguish its taskbar grouping and shortcut identity from the newer desktop application.
- Updates the source version to `36.10.7` and the package version to `0.2.7`.

## 中文

- 浏览器文档标题和 Electron 窗口标题均标记为 `Gaussian Scene Workbench (Legacy HTML)`。
- 使用独立的 Windows AppUserModelID `Fubukisama.GaussianSceneWorkbench.LegacyHtml`，使 Windows 能将其任务栏分组和快捷方式身份与新版桌面应用区分开。
- 源码版本更新至 `36.10.7`，封装版本更新至 `0.2.7`。

## 日本語

- ブラウザーの文書タイトルと Electron ウィンドウタイトルの両方を `Gaussian Scene Workbench (Legacy HTML)` として識別します。
- 専用の Windows AppUserModelID `Fubukisama.GaussianSceneWorkbench.LegacyHtml` を使用し、タスクバーのグループとショートカットを新しいデスクトップアプリから区別できるようにします。
- ソースバージョンを `36.10.7`、パッケージバージョンを `0.2.7` に更新します。
